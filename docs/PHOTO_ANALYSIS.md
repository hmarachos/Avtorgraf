# Анализ фотографий строительных нарушений

## Обзор

Система АВТОГРАФ поддерживает анализ фотографий строительных площадок для выявления нарушений и несоответствий с возможностью **фокусировки на конкретных элементах**.

## Архитектура

### 3-шаговый процесс

1. **ШАГ 1: Анализ фотографии с учетом запроса пользователя**
   - Vision модель анализирует изображение с акцентом на указанный элемент
   - Приоритет: котлованы, откосы, конкретные конструкции
   - Если запрошенный элемент не виден - модель явно об этом сообщает
   - Формирует детальное описание всех найденных проблем

2. **ШАГ 2: Формирование специфического поискового запроса**
   - На основе описания создается запрос с акцентом на тему
   - Для котлованов: "угол откосов", "крепление стенок", "расстояние от бровки"
   - Запрос оптимизирован для поиска в базе знаний РБ

3. **ШАГ 3: Генерация ответа с нормативными ссылками**
   - LightRAG ищет релевантные нормативные документы
   - Формируется ответ: выявленные нарушения + методы устранения
   - Сначала описание по запрошенной теме, потом дополнительные нарушения

## Конфигурация

### Переменные окружения

```env
# Vision модель для анализа фотографий (OpenRouter)
OPENROUTER_API_KEY=sk-or-v1-your-key
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
VISION_MODEL=google/gemini-2.0-flash-exp:free
```

### Поддерживаемые модели

- `google/gemini-2.0-flash-exp:free` (рекомендуется, бесплатно)
- `openai/gpt-4o-mini`
- `anthropic/claude-3.5-sonnet`
- Любая multimodal модель через OpenRouter

## API

### POST /api/analyze-photo

Анализирует фотографию и возвращает отчет о нарушениях.

**Request:**
```json
{
  "image": "data:image/jpeg;base64,/9j/4AAQ...",
  "format": "jpeg",
  "session_id": "optional-session-id"
}
```

**Response:**
```json
{
  "session_id": "abc123",
  "answer": "## Выявленные нарушения\n\n...\n\n## Методы устранения\n\n...",
  "references": [
    {
      "title": "СН 1.03.01-2019",
      "source": "...",
      "snippet": "..."
    }
  ],
  "elapsed_ms": 3500
}
```

## Фронтенд

### Загрузка фото с текстовым запросом

1. Нажмите на иконку фото в поле ввода
2. Выберите изображение (JPEG, PNG, WebP)
3. Максимальный размер: 10MB
4. Отобразится превью фото
5. **В поле ввода укажите, на что обратить внимание** (например: "какие нарушения при устройстве откосов котлована?")
6. Нажмите "Отправить" для анализа

### Примеры специфических запросов

- "какие нарушения при устройстве откосов котлована?"
- "проверь крепление стенок котлована"
- "оцени качество бетонирования фундамента"
- "есть ли нарушения безопасности при работе на высоте?"
- "правильно ли складированы материалы?"

**Важно:** Чем конкретнее запрос, тем точнее анализ!

### Визуальное представление

- Фото пользователя отображается в чате с миниатюрой
- Ответ ассистента содержит две секции:
  - **Выявленные нарушения** - описание проблем
  - **Методы устранения** - рекомендации с нормативными ссылками

## Категории нарушений

Vision модель ищет:

1. **Нарушения технологии производства работ**
   - Неправильное выполнение процессов
   - Отступления от проекта
   - Нарушения последовательности

2. **Нарушения требований безопасности**
   - Отсутствие средств защиты
   - Опасные зоны без ограждений
   - Нарушения при работе на высоте

3. **Дефекты строительных конструкций**
   - Трещины, сколы, раковины
   - Коррозия арматуры
   - Нарушения гидроизоляции

4. **Несоответствия проектной документации**
   - Отклонения от размеров
   - Несогласованная замена материалов
   - Несоответствие конструкций

5. **Нарушения при складировании материалов**
   - Неправильное хранение
   - Отсутствие поддонов/укрытий
   - Смешивание материалов

## Ограничения

- Vision модель не заменяет эксперта-строителя
- Рекомендуется консультация с юристом для критических случаев
- Ответы основаны только на данных из LightRAG

## Безопасность

- Изображения обрабатываются через OpenRouter API
- Фотографии не сохраняются на сервере
- Данные передаются в зашифрованном виде (HTTPS)
