# 📚 Гайд по ссылкам на документы в АВТОГРАФ

## 🎯 Важность ссылок

Каждый ответ в АВТОГРАФ **должен содержать ссылки на конкретные пункты нормативных документов**. Это обеспечивает:

✅ Юридическую обоснованность  
✅ Возможность проверки информации  
✅ Ясность источника  
✅ Доверие специалистов  

## 📋 Обязательный формат ссылок

### Синтаксис

```markdown
**[Название документа, пункт/раздел]**
```

### Примеры правильных ссылок

| Тип | Примеры |
|-----|---------|
| **СН (Строительные Нормы)** | `**[СН 1.03.01-2019, п. 3.4]**` |
| | `**[СН 2.01.01-98, раздел 2]**` |
| **СТБ (Межгосударственные стандарты)** | `**[СТБ 1937-2022, п. 4.1.2]**` |
| | `**[СТБ EN 206-2020, раздел 5]**` |
| **Инструкции** | `**[Инструкция 1-2020, п. 6.3]**` |
| **Кодексы и законы** | `**[Кодекс Республики Беларусь о земле, ст. 15]**` |
| **ТКП (Технические кодексы)** | `**[ТКП 45.13330.12, п. 7.2]**` |

## ✅ Правильный формат ответа

### ❌ НЕПРАВИЛЬНО

```
Согласно нормативам нужны следующие документы:
- Проектная документация
- Рабочая документация
- Смета затрат

Кроме того, нужно соблюдать требования по хранению материалов.
```

**Проблемы:**
- Нет ссылок на документы
- Невозможно проверить информацию
- Не ясно какие нормы применяются

### ✅ ПРАВИЛЬНО

```
## Документы для подписания АОСР

Согласно **[СН 1.03.01-2019, п. 3.2.1]** и **[СТБ 1937-2022, раздел 2]** 
для подписания акта освидетельствования скрытых работ необходимо подготовить:

### Обязательные документы

- **Проектная документация** — согласно **[СН 1.03.01-2019, п. 2.1.4]**
- **Рабочая документация** — согласно **[СН 1.03.01-2019, п. 2.1.5]**
- **Смета затрат** — если требуется по **[СН 1.03.01-2019, п. 3.1]**
- **Акт приемки работ** — в соответствии с **[ГОСТ 13055-2022, п. 4.2]**

### Требования к хранению

Согласно **[СТБ 1937-2022, раздел 4]** строительные материалы должны 
храниться в упакованном и защищенном состоянии на соответствующих поддонах 
или в контейнерах.

### Процедура контроля

Согласно **[СН 1.05.01-2019, п. 4.3.2]** авторский надзор должен:
1. Осмотреть скрытые работы до их закрытия
2. Документировать результаты в журнале
3. Подписать акт только при соответствии норме
```

**Преимущества:**
- Каждое утверждение имеет ссылку
- Легко проверить информацию
- Ясны применяемые нормы

## 📖 Структура ответа с ссылками

### 1️⃣ Заголовок вопроса

```markdown
## Какие документы нужны для подписания АОСР?
```

### 2️⃣ Основной ответ с ссылками

```markdown
Согласно **[СН 1.03.01-2019, п. 3.2.1]** необходимо подготовить 
следующие документы...
```

### 3️⃣ Детальное описание с подссылками

```markdown
### Пункт 1

- **Пункт 1.1** — согласно **[СН 1.03.01-2019, п. 3.2.1]**
- **Пункт 1.2** — в соответствии с **[СТБ 1937-2022, раздел 2]**
```

### 4️⃣ Рекомендации или дополнение

```markdown
### Важно помнить

Согласно **[СН 1.05.01-2019, п. 2.1]** авторский надзор должен 
контролировать соответствие работ проектной документации.
```

## 🔍 Как читать и использовать ссылки

### Когда видите ссылку

```
**[СН 1.03.01-2019, п. 3.4]**
```

Это значит:
- **СН 1.03.01-2019** — Строительные Нормы 1.03.01 от 2019 года
- **п. 3.4** — пункт 3.4 этого документа
- На эту ссылку можно **найти и проверить** в документе

### Как найти документ

1. **На сайте** www.snip.by или www.normativ.by
2. **В библиотеке** вашей организации
3. **У коллег** в строительной компании
4. **В интернете** через поиск "СН 1.03.01-2019 PDF"

## 🎓 Примеры полных ответов

### Пример 1: Вопрос о документах

**Вопрос:**
> Какие документы должны быть на строительной площадке?

**Ответ:**

```
## Документы на строительной площадке

Согласно **[СН 1.03.01-2019, п. 5.1]** и **[Инструкция 1-2020, п. 3.1]** 
на строительной площадке должны находиться:

### Обязательные документы

- **Лицензия** на выполнение работ — согласно **[СН 1.03.01-2019, п. 1.2]**
- **Проектная документация** — согласно **[СН 1.03.01-2019, п. 2.1]**
- **Журналы контроля** — согласно **[СН 1.05.01-2019, п. 2.1]**
- **Сертификаты материалов** — согласно **[СТБ 1937-2022, п. 4.1]**
- **Акты освидетельствования** — согласно **[СН 1.03.01-2019, п. 3.4]**

### Ответственное лицо

Согласно **[СН 1.03.01-2019, п. 5.1.2]** все документы должны быть 
на рабочем месте представителя авторского надзора.
```

### Пример 2: Вопрос о дефектах

**Вопрос:**
> Можно ли подписать АОСР при наличии дефектов?

**Ответ:**

```
## Подписание АОСР при дефектах

### Ответ: НЕТ, подписывать нельзя

Согласно **[СН 1.03.01-2019, п. 3.4.1]** акт освидетельствования 
скрытых работ может быть подписан **только при полном соответствии** 
выполненных работ проектной документации и нормативным требованиям.

### Что делать при обнаружении дефектов

1. **Зафиксировать дефект** в журнале авторского надзора согласно 
   **[СН 1.05.01-2019, п. 2.2]**
2. **Требовать исправления** согласно **[СН 1.03.01-2019, п. 6.1]**
3. **Проверить исправления** перед подписанием согласно 
   **[СН 1.03.01-2019, п. 3.4.2]**
4. **Только после исправления** подписать акт согласно 
   **[СН 1.03.01-2019, п. 3.4.3]**

### Ответственность

Согласно **[СТБ 1937-2022, раздел 3]** авторский надзиратель несет 
ответственность за качество контроля. Подписание акта при наличии 
дефектов — это нарушение.
```

## ⚡ Быстрая справка

### Главное правило
```
🔑 КАЖДОЕ УТВЕРЖДЕНИЕ = КОНКРЕТНЫЙ ПУНКТ ДОКУМЕНТА
```

### Формула правильного ответа
```
Утверждение + **[Документ, пункт]** = Юридически обоснованный ответ
```

### Что ВСЕГДА должно быть в ответе
✅ Наименование документа (СН, СТБ, инструкция)  
✅ Номер пункта или раздела  
✅ Год издания документа (если актуально)  
✅ Ясность источника информации

## 📞 Если вы не уверены

Если в ответе видите фразу:
```
"В базе знаний подтверждение не найдено"
```

Это означает что система не нашла нужный документ. В этом случае:
1. Проверьте формулировку вопроса
2. Переформулируйте вопрос
3. Проверьте в документах самостоятельно
