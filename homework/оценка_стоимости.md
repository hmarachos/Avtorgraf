# Оценка стоимости проекта АВТОГРАФ
## Период: 12 месяцев (июнь 2026 — май 2027)

---

## 1. Контекст и допущения

**Тип проекта:** MVP ИИ-ассистента для специалистов строительного надзора РБ  
**Команда:** 1 разработчик full-stack + периодически аутсорс  
**Аудитория:** 10–50 активных пользователей (проектные институты РБ)  
**Инфраструктура:** Облачный сервер + локально развёрнутый LightRAG  
**Валюта:** USD (основные затраты) с конвертацией в BYN по курсу ~3,25 руб./USD

---

## 2. Расходы на LLM (OpenAI API)

LightRAG использует OpenAI GPT-4o для:
- Построения графа знаний (индексирование документов) — разовая операция
- Ответов на запросы пользователей — постоянные расходы

### 2.1 Разовое индексирование базы знаний

| Параметр | Значение |
|---|---|
| Объём документов | ~125 документов, ~500 000 токенов |
| Модель | GPT-4o |
| Цена input | $2,50 / 1M токенов |
| Цена output | $10,00 / 1M токенов |
| Input (извлечение сущностей) | ~500K × $2,50 = **$1,25** |
| Output (генерация графа) | ~200K × $10,00 = **$2,00** |
| **Итого разово** | **~$3–5** |

### 2.2 Ежемесячные запросы пользователей

Допущения:
- 30 активных пользователей × 10 запросов/день × 22 рабочих дня = 6 600 запросов/мес.
- Средний запрос: ~800 токенов input + ~1 200 токенов output

| Параметр | Расчёт | Сумма/мес. |
|---|---|---|
| Input токены | 6 600 × 800 = 5,28M токенов × $2,50/1M | **$13,20** |
| Output токены | 6 600 × 1 200 = 7,92M токенов × $10,00/1M | **$79,20** |
| **Итого/мес.** | | **~$92** |
| **Итого/год** | $92 × 12 | **~$1 104** |

> **Примечание:** При использовании GPT-4o mini ($0,15/1M input, $0,60/1M output) стоимость снизится примерно в 10 раз — до ~$110/год. Рекомендуется протестировать качество на тестовых кейсах.
